# Roque Imóveis — Landing Page

Vite + React + TypeScript + Tailwind. Pronto para deploy no Vercel.

## Desenvolvimento

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy no Vercel

1. Suba o repositório no GitHub.
2. Importe no Vercel (framework detectado: Vite).
3. Configure a variável `VITE_WHATSAPP_NUMBER` (opcional).
4. Deploy automático.
